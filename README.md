# Proposal Digital HUT RI ke-81 Dusun Maguan (2026)

Repositori ini berisi aplikasi Flutter sederhana yang menampilkan draf digital untuk **Proposal Kegiatan Peringatan HUT Kemerdekaan RI ke-81 Tahun 2026 di Dusun Maguan**, yang diselenggarakan oleh **Paguyuban Angkatan Muda Maguan**.

## Fitur
- Tampilan desain proposal yang bersih, responsif, dan elegan.
- Berisi draf resmi "Latar Belakang" acara.
- Mendukung tampilan mobile, tablet, dan desktop/web.

## Cara Menjalankan
1. Pastikan Anda telah menginstal Flutter di perangkat Anda.
2. Jalankan perintah `flutter pub get` untuk mengunduh dependensi.
3. Jalankan aplikasi menggunakan `flutter run`.
